## How to install

Create a venv, install the dependencies

```
python3 -m venv /tmp/clivandegentenaar
source /tmp/clivandegentenaar/bin/activate
pip install -r requirements.txt
```

## How to run

Run the application:

```
python app.py
```

Run cli

```
npx tsc
chmod -R 777 ./build
./build/cli.js search <query>
```
